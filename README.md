# Pangur

Pangur is a copyist's cat (a copycat),
a utility to copy and mirror directory trees.

[Pangur Bán] is a poem written in Old Irish by a copyist (scribe) about his cat.
Enjoy [Seamus Heaney's translation].

[Seamus Heaney's translation]: https://www.poetryfoundation.org/poetrymagazine/poems/48267/pangur-ban
